# 🔐 戰盟圈子 - 微信 OAuth 登录

点击下方按钮一键部署到 Vercel：

[![Deploy to Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/zhanmeng001/wechat-oauth)

部署完成后：
1. 复制 Vercel 给你的域名（`xxx.vercel.app`）
2. 去微信公众号后台 → **设置与开发 → 开发与接口管理**
3. 设置 **网页授权域名** 为你的 Vercel 域名
4. 把 `https://你的域名/api/auth` 加为 **回调域名**

然后在戰盟圈子里点「💬 微信登录」就能自动获取昵称和头像了！
